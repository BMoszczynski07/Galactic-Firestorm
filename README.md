## What is Galactic Firestorm?

Galactic Firestorm is a game written in python (pygame library). In this game you are a trailblazer who owns a ship. Your task is to defeat all the enemies coming near you.

## Ship steering

A key -> to the left
D key -> to the right
Space -> shot
